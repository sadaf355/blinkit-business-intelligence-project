# 🛒 Blinkit Business Intelligence & Analytics Project

## 📌 Project Overview

This project is an end-to-end Business Intelligence solution developed using **Excel, PostgreSQL, SQL, and Power BI**. It simulates a real-world analytics workflow for Blinkit, an online grocery delivery platform, transforming raw operational data into meaningful business insights.

The project demonstrates the complete data analytics lifecycle, including:

- Data Cleaning using Excel
- Database Design using PostgreSQL
- Business Analytics using SQL
- Interactive Dashboard Development using Power BI
- Business Documentation and KPI Reporting

The objective is to help stakeholders monitor business performance, analyze customer behavior, evaluate marketing effectiveness, and optimize operational efficiency through interactive dashboards.

---

# 🎯 Project Objectives

The primary objectives of this project are:

- Analyze overall business performance
- Understand customer purchasing behavior
- Identify top-performing products and categories
- Measure marketing campaign effectiveness
- Evaluate customer satisfaction
- Monitor delivery performance
- Analyze inventory health
- Generate actionable business recommendations

# 📌 Key Business Insights

Some major insights obtained from the dashboard include:

- Blinkit generated over ₹11 Million in revenue during the analysis period.
- Repeat customers contributed significantly to overall revenue.
- Certain product categories consistently outperformed others in both revenue and margin.
- Marketing campaigns delivered varying levels of return on investment (ROAS).
- Customer ratings remained relatively stable throughout the year.
- Delivery efficiency directly impacted customer satisfaction.
- Inventory damage rates varied across product categories.
